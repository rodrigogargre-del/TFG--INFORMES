
# Códigos desarrollados – Rota

Esta carpeta contiene los bloques de código desarrollados para el análisis y evaluación de ejercicios de navegación en el entorno de Rota.

Aquí se incluyen los cuadernos empleados durante el desarrollo del trabajo para procesar los datos, representar las derrotas, detectar eventos críticos y evaluar el ejercicio conforme a los criterios definidos.

Dentro de esta carpeta se incorporan los códigos relacionados con la evaluación de la seguridad en la navegación y la valoración de encuentros según el RIPA.
