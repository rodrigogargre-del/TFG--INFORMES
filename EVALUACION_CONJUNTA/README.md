# Evaluación conjunta de ejercicios de navegación

Esta carpeta contiene cuatro ejercicios representativos utilizados para mostrar el funcionamiento completo del sistema de evaluación desarrollado en este trabajo.

Para cada entorno se incluyen dos ejercicios, permitiendo observar el comportamiento del sistema ante diferentes situaciones.

Cada ejercicio incluye dos informes generados automáticamente por el evaluador, en el que se analiza el desarrollo de la navegación del buque propio y los contactos detectados durante el ejercicio.

Esta sección tiene como finalidad mostrar de forma integrada cómo el sistema procesa los datos de navegación, identifica situaciones relevantes y genera informes interpretativos que permiten valorar el nivel de seguridad del ejercicio realizado.
